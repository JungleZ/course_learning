# Satya Poetry Collection

This repository is intended to store the poetry collection of Satya (Virginia Satyr).

## Project Structure
- `poems/` - Directory containing individual poetry files in Markdown format
- `README.md` - This file
- `CONTRIBUTING.md` - Guidelines for contributing to the collection
- `LICENSE` - License information (to be added)

## Usage
Add poetry files to the `poems/` directory with `.md` extension.
Each file should contain one poem or a collection of related poems.

## Naming Convention
Use descriptive filenames for poetry files, for example:
- `love-and-family.md`
- `healing-poems.md`
- `daily-reflections.md`

## Note
Please ensure you have the right to distribute any copyrighted poetry before adding it to this collection.
