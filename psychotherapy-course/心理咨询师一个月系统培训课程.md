# 心理咨询师一个月系统培训课程

## 课程概述

本课程旨在培养具备多流派整合能力的心理咨询师，通过系统学习CBT、NLP、精神分析、萨提亚家庭治疗、家庭系统排列和艾瑞克森催眠六大核心技术，掌握从接待来访到完整治疗的全流程技能。课程分为四个阶段，循序渐进，最终实现多流派融合应用。催眠技术将作为体验性改变的重要工具，与NLP、萨提亚等技术形成互补。

## 第一周：基础理论与来访者接待

### 第一天：咨询伦理入门

**上午（2小时）：**

- 学习心理咨询伦理的核心原则
- 阅读《中国心理学会伦理守则》重点章节
- 整理笔记：保密原则、双重关系、知情同意三大核心

**下午（2小时）：**

- 案例讨论：伦理困境分析（3个模拟场景）
- 绘制伦理决策流程图
- **行动作业**：列出你在咨询中可能遇到的5个伦理问题

**晚间（1小时）：**

- 反思自己过往经历中的边界经历

## 技术脉络总结

### 接待来访者流程（按天）

| 阶段 | 关键任务 | 对应天数 |
|------|----------|----------|
| 预约 | 电话接听、问题收集、判断转介 | 第2天 |
| 初诊 | 知情同意、保密说明、自我介绍 | 第2天 |
| 评估 | 收集资料、量表测评、概念化 | 第3天 |
| 计划 | 目标设定、技术选择、方案制定 | 第23天起 |
| 治疗 | 技术实施、过程评估、调整 | 第24天起 |
| 结束 | 回顾总结、分离处理、后续跟进 | 第28天 |

### 核心技术框架（按天）

**CBT技术链：**

- 第4天：认知概念化
- 第6天：苏格拉底提问
- 第8天：认知重构
- 第9天：行为实验
- 第5天：行为激活
- 第24天：CBT综合练习

**NLP技术链：**

- 第10天：NLP基础+感知位置
- 第11天：重新框架+锚定

**艾瑞克森催眠技术链（催眠3.0）：**

- 第13天：隐喻故事+阻抗利用+双重绑定
