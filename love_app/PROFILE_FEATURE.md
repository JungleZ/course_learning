# 👤 个人资料功能说明

## 🎯 功能概述

为等公交聊天室添加了完整的个人资料系统，用户可以创建和管理自己的详细档案，包括基本资料、兴趣爱好、价值观、生活方式等。所有数据使用浏览器localStorage本地存储，保护用户隐私。

---

## ✨ 核心功能

### 1. **完整个人资料表单**
- ✅ 头像表情选择（12种emoji）
- ✅ 基本资料（昵称、年龄、性别、城市、职业、简介）
- ✅ 兴趣标签管理（最多10个）
- ✅ 价值观与生活观滑块
- ✅ 重要价值观多选
- ✅ 生活方式设置
- ✅ 期望与目标填写

### 2. **本地存储（localStorage）**
- ✅ 自动保存所有资料
- ✅ 页面刷新后自动加载
- ✅ 完全隐私，不上传服务器
- ✅ 跨会话持久化

### 3. **智能交互**
- ✅ 实时预览功能
- ✅ 表单验证
- ✅ 动态标签添加/删除
- ✅ 滑块值实时显示
- ✅ 成功提示通知

### 4. **用户体验**

## 🚀 使用流程

### 步骤1: 访问个人资料页面

**方式1: 从聊天室进入**
1. 在聊天室右上角点击"我的资料"按钮
2. 直接进入个人资料编辑页面

**方式2: 直接访问**
```
http://localhost:5000/profile
```

### 步骤2: 填写个人资料

#### 2.1 选择头像
- 点击喜欢的emoji表情
- 选中的表情会高亮显示
- 顶部大头像实时更新

#### 2.2 填写基本信息
- 输入昵称（必填）
- 选择年龄、性别
- 填写城市、职业
- 写一段个人简介

#### 2.3 添加兴趣标签
- 在输入框中输入兴趣
- 按Enter键添加
- 点击标签上的×删除
- 最多添加10个

#### 2.4 调整价值观滑块
- 拖动滑块调整比例
- 实时显示百分比
- 根据真实情况设置

#### 2.5 选择重要价值观
- 勾选符合自己的价值观
- 可多选
- 至少选择1-3个

#### 2.6 设置生活方式
- 选择作息时间
- 选择饮食习惯
- 选择运动频率
- 填写学习目标（可选）

#### 2.7 填写期望与话题
- 描述想认识什么样的人
- 勾选想聊的话题
- 帮助匹配志同道合的朋友

### 步骤3: 预览资料

**点击"预览资料"按钮：**
- 查看完整的资料展示
- 确认信息是否准确
- 检查格式是否正确
- 可以在预览中直接保存

### 步骤4: 保存资料

**点击"保存个人资料"按钮：**
- 系统验证必填项
- 保存到localStorage
- 显示成功提示
- 2秒后自动返回聊天室

### 步骤5: 后续修改

**随时可以修改：**
- 再次进入个人资料页面
- 之前填写的内容会自动加载
- 修改后重新保存即可
- 覆盖之前的数据

## 💾 数据存储说明

### 存储位置
```
浏览器 localStorage
Key: busChatProfile
Value: JSON字符串
```

### 数据结构示例
```json
{
  "avatar": "😎",
  "nickname": "阳光少年",
  "age": "25",
  "gender": "male",
  "city": "北京",
  "occupation": "程序员",
  "bio": "热爱编程，喜欢旅行",
  "tags": ["编程", "旅行", "摄影", "音乐"],
  "workLifeBalance": 60,
  "spendingHabit": 40,
  "socialPreference": 70,
  "values": ["诚实", "成长", "自由"],
  "sleepSchedule": "normal",
  "dietHabit": "healthy",
  "exerciseFrequency": "often",
  "learningGoal": "学习人工智能",
  "expectation": "认识志同道合的朋友",
  "topics": ["科技", "旅行", "音乐"],
  "createdAt": "2026-04-28T12:00:00.000Z",
  "updatedAt": "2026-04-28T12:30:00.000Z"
}
```

### 存储特点
✅ **本地存储** - 数据只在用户浏览器中  
✅ **持久化** - 关闭浏览器后数据仍保留  
✅ **隐私安全** - 不上传到服务器  
✅ **跨会话** - 下次访问自动加载  
✅ **容量充足** - localStorage通常5-10MB  

### 清除数据
**方法1: 浏览器设置**
- Chrome: 设置 → 隐私和安全 → 清除浏览数据
- 选择"Cookie和其他网站数据"
- 清除后会丢失个人资料

**方法2: JavaScript控制台**
```javascript
localStorage.removeItem('busChatProfile');
```

**方法3: 开发者工具**
- F12打开开发者工具
- Application → Local Storage
- 找到`busChatProfile`并删除

## 🔒 隐私与安全

### 隐私保护

✅ **本地存储** - 数据不离开用户设备  
✅ **匿名使用** - 不需要注册账号  
✅ **自主控制** - 随时可以删除  
✅ **无追踪** - 不收集使用数据  

### 安全建议

⚠️ **不要填写敏感信息**
- 真实姓名
- 电话号码
- 身份证号
- 详细地址
- 财务信息

⚠️ **公共电脑注意清除**
- 使用完后清除localStorage
- 或使用无痕模式

⚠️ **定期更新资料**
- 保持信息时效性
- 删除过时内容
