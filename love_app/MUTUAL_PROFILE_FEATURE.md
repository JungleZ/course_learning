# 👥 互相查看个人资料功能说明

## 🎯 功能概述

在等公交聊天室中，当两个用户聊天超过5分钟后，可以互相查看对方的个人资料，包括微信号等联系方式。这个功能鼓励深入交流，同时保护用户隐私。

---

## ✨ 核心功能

### 1. **时间限制机制**
- ✅ 必须聊天满5分钟（300秒）才能查看资料
- ✅ 实时显示剩余时间提示
- ✅ "可加好友"徽章提示
- ✅ 未满5分钟时显示倒计时

### 2. **查看资料按钮**
- ✅ 在其他用户的消息旁边显示"查看资料"按钮
- ✅ 自己的消息不显示该按钮
- ✅ 点击后检查聊天时长
- ✅ 美观的按钮样式和悬停效果

### 3. **个人资料展示**
- ✅ 模态框展示对方资料
- ✅ 显示昵称、用户ID
- ✅ 微信号展示（如果对方已设置）
- ✅ 隐私保护提示

### 4. **隐私保护**
- ✅ 微信号仅在个人资料中设置后可见
- ✅ 需要双方都完善资料
- ✅ 时间门槛防止骚扰
- ✅ 用户自主控制信息公开

---

## 🚀 使用流程

### 步骤1: 开始聊天
1. 进入聊天室 http://localhost:5000/bus-chat
2. 输入昵称加入聊天
3. 与其他用户开始聊天

### 步骤2: 等待5分钟
- 系统自动追踪聊天时长
- 右上角显示聊天时长计时器
- 满5分钟后显示"可加好友"徽章

### 步骤3: 查看对方资料

**方式1: 从消息列表查看**
1. 找到对方的任意一条消息
2. 点击消息上方的"查看资料"按钮
3. 系统检查是否满5分钟
4. 如果未满，显示剩余时间提示
5. 如果已满，显示对方资料模态框

**方式2: 主动完善自己的资料**
1. 点击右上角"我的资料"
2. 完善个人资料，包括微信号
3. 保存后，对方也能看到你的资料

### 步骤4: 交换联系方式

**在资料模态框中：**
1. 查看对方的微信号（如果已设置）
2. 点击"复制联系方式"按钮
3. 或者截图保存
4. 添加微信好友

### 步骤5: 继续交流
- 可以在微信上继续聊天
- 也可以回到聊天室继续互动
- 建立更深入的友谊

## 📋 技术实现

### 前端实现

#### 1. 聊天时长追踪
```javascript
function getChatDuration() {
    if (!chatStartTime) return 0;
    const now = new Date().getTime();
    const start = new Date(chatStartTime).getTime();
    return Math.floor((now - start) / 1000); // 返回秒数
}
```

#### 2. 查看资料检查
```javascript
function viewUserProfile(targetUserId, targetUsername) {
    const chatDuration = getChatDuration();
    
    if (chatDuration < 300) {
        const remainingTime = 300 - chatDuration;
        const minutes = Math.floor(remainingTime / 60);
        const seconds = remainingTime % 60;
        
        showNotification(
            `还需聊天 ${minutes}分${seconds}秒 才能查看资料`, 
            'warning'
        );
        return;
    }
    
    // 显示对方资料
    showOtherUserProfile(targetUserId, targetUsername);
}
```

### 后端API

#### 获取用户信息
```python
@app.route('/api/chat/user-profile', methods=['GET'])
def get_user_profile():
    """获取用户的个人资料"""
    user_id = request.args.get('user_id')
    
    if user_id and user_id in online_users:
        user_info = online_users[user_id]
        return jsonify({
            'success': True,
            'user_id': user_id,
            'username': user_info['username'],
            'join_time': user_info['join_time']
        })
    
    return jsonify({'success': False, 'error': '用户不存在'}), 404
```

## 🔒 隐私与安全

### 隐私保护措施

✅ **时间门槛** - 5分钟聊天防止随意查看  
✅ **自主选择** - 用户可以选择不设置微信号  
✅ **本地存储** - 资料不上传服务器  
✅ **随时修改** - 可以随时修改或删除资料  
✅ **匿名参与** - 不需要真实身份  

### 安全建议

⚠️ **谨慎分享微信号**
- 确认对方值得信任
- 不要在公开场合分享
- 定期清理陌生好友

⚠️ **保护个人信息**
- 不要填写过于详细的地址
- 不要填写财务信息
- 不要填写身份证号

⚠️ **注意网络安全**
- 警惕诈骗行为
- 不要轻易转账
- 见面选择公共场所

### 时间阈值配置

**当前设置：** 5分钟（300秒）

**修改方法：**
在 `bus_chat.js` 中修改：
```javascript
if (chatDuration < 300) { // 改为其他秒数
    // ...
}
```

**建议值：**
- 快速交友: 3分钟（180秒）
